# Portfolio V3
This is version 3 of my portfolio, built with Next.js, Tailwind CSS, and TypeScript.

## Tech Stack
- Next.js
- Tailwind CSS
- TypeScript
- React
- Motion(.dev)
- React Query
- OpenMeteo API
- Vercel
